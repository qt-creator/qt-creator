Qt Creator supports the model context protocol (MCP) to let AI assistants
control it for debugging, building, and managing projects.

To start the Qt Creator MCP server and use an MCP client, install a client and
activate the Qt Creator MCP server extension.

Qt Creator starts the MCP server when you activate the Qt Creator MCP server
extension. The server runs on the computer and accepts connections on an
automatically set port.

### 1. Activate the extension

Activate the **Qt Creator MCP Server** extension. Optionally, also activate
**ACP Server** if you want to use AI agents to trigger Qt Creator actions such as
building or running your project.

### 2. View Qt Creator MCP server status

Go to **Preferences > AI > Qt Creator MCP Server**.

You can change the IP address and port number of the server.
